# GOROSKOPE

Минимальный Telegram-бот и Tkinter UI для проверки OpenAI.

1. Заполните `.env` через `run_ui.bat` (токен бота, `OPENAI_API_KEY`, модель, `ADMIN_TELEGRAM_ID`).
2. Создайте виртуальное окружение `.venv` (опционально) и установите зависимости: `pip install -r requirements.txt`.
3. Запускайте настройщик двойным кликом `run_ui.bat`, а бота — `run_bot.bat` или кнопкой **Run Bot** в UI.
